# Sample Flow

## Prompt
"We're close to release on a university web system. Stakeholders say quality is fine, but defect rates are rising, change requests are slow to implement, and nobody can explain the tradeoff between maintainability and speed of delivery."

## Suggested flow
1. quality-attribute-model-builder
2. quality-weighting-reviewer
3. quality-goal-scorer
4. technical-debt-auditor
5. maintainability-reviewer
6. process-quality-checker
7. product-vs-process-quality-splitter
8. quality-evidence-summarizer
9. quality-risk-register-builder

## Expected artifacts
- explicit quality attribute model
- weighting matrix with tradeoff rationale
- goal scorecard with confidence notes
- technical debt register
- maintainability review
- process quality findings
- evidence summary
- quality-focused risk register
